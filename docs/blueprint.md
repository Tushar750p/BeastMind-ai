# **App Name**: BeatMind AI

## Core Features:

- AI Text-to-Music Tool: Convert descriptive prompts into original music compositions using a reasoning-based generative tool.
- Configurable Music Duration: Choose specific lengths for the generated background music, such as 15s or 30s clips.
- High-Fidelity Preview Player: Stream the generated AI music immediately within a modern, wave-responsive audio player.
- One-Click MP3 Export: Download generated audio tracks instantly to your local storage for project use.
- Genre & Mood Pre-settings: UI selection chips for quick prompt refinement such as Cinematic, Gaming, or Lo-Fi styles.
- Cloud Generation Library: Save generated tracks and prompts to a Firestore database for cross-device accessibility.
- Real-time Queue Status: Visual indicator for track processing status to monitor AI generation in real-time.

## Style Guidelines:

- Primary color: Electric Violet (#9747FF) to represent high-tech creative energy against a dark interface.
- Background color: Deep Charcoal (#0F0E13) for a 'Night Mode' studio atmosphere that enhances visual focus.
- Accent color: Deep Ocean Blue (#0094FF) to distinguish playback controls and active interaction states.
- Headline font: 'Space Grotesk', a geometric sans-serif that lends a technical, modern feel; paired with 'Inter' for body text to ensure maximum readability.
- Clean, monoline icons that focus on sonic architecture, such as frequency waves, sound bars, and minimal play/pause shapes.
- Centered, single-page dashboard focusing on the prompt entry bar as the visual focal point, surrounded by negative space.
- Subtle breathing pulses on the generation button and smooth frequency visualizers during music playback.
